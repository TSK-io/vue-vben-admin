根据你的需求文档（守护桑榆——反电诈AI助手），以下按各核心模块推荐满足需求的开源项目，均不使用 Java。

***

## 整体框架基座（RBAC + 多角色 + 管理后台）

**推荐：`django-vue3-admin`**

这是一个完全开源的快速开发平台，基于 Python Django + DRF + Vue3 + Element Plus + TypeScript 构建 。它内置了细粒度 RBAC 权限体系（支持按列级别控制），支持动态权限菜单加载、多角色注册、JWT 多端认证，与文档中老年用户、子女、社区、管理员四种角色的差异化菜单需求完全匹配 。 [github](https://github.com/huge-dream/django-vue3-admin)

**备选：`fastapi-vue-admin`（SenQi-666）**

基于 FastAPI + Vue3 + Ant Design Vue + SQLAlchemy 2.0，同样支持 RBAC 按钮级权限和数据级权限控制，适合追求高性能异步 API 的团队 。 [github](https://github.com/SenQi-666/fastapi-vue-admin)

***

## 通话风险识别（语音转文字 + AI 分析）

**推荐：`openai/whisper`**

OpenAI 开源的 Whisper 模型支持多语言（含普通话和方言）、离线运行，适合对老年人通话内容做实时语音转文字，然后交给 NLP 模型做风险分析 。已有基于 Whisper + FastAPI + Scikit-learn 的开源诈骗电话检测原型项目，集成了关键词检测、风险打分（低/中/高）和语音特征分析 。 [linkedin](https://www.linkedin.com/posts/joel-jacob-roji-a10792315_github-joeljacobrojivoicephishdetection-activity-7434815424007057409-x7JQ)

**配套数据集：`TeleAntiFraud-28k`**

首个专门面向电信诈骗的开源音频-文本数据集，包含 28,511 个语音-文本对（307 小时），涵盖诈骗场景分类和诈骗类型标注，可用于微调识别模型 。 [github](https://github.com/JimmyMa99/TeleAntiFraud)

***

## 短信风险识别（NLP 文本分类）

**推荐：Hugging Face `transformers` + 中文预训练模型**

使用 `bert-base-chinese` 或 `chinese-roberta-wwm-ext` 作为基础模型，结合 Hugging Face `transformers` 库，可以快速训练一个短信诈骗分类器，识别冒充身份、中奖诱导、验证码套取等风险模式。该方案为纯 Python 实现，可与 FastAPI 后端无缝集成。

***

## 实时告警与通知系统

**推荐：FastAPI + Redis Streams + WebSocket**

该架构完全开源，通过 Redis Stream 实现消息的持久化排队，再通过 WebSocket 实时推送给子女端 。流程是：风险事件写入 Redis Stream → Consumer 消费并存入 PostgreSQL → WebSocket 服务端推送告警到浏览器，满足文档中"高风险事件优先推送子女和社区"的需求 。 [dev](https://dev.to/geetnsh2k1/building-a-real-time-notification-service-with-fastapi-redis-streams-and-websockets-52ib)

**备选：Django Channels**

若整体框架选 Django，可使用 Django Channels 原生支持 WebSocket，实现低延迟推送 。 [app-generator](https://app-generator.dev/docs/technologies/fastapi/real-time-notifications.html)

***

## 工单与协同管理

**推荐：`paw`（Python 开源工单系统）**

纯 Python（Django 构建）的开源工单管理系统，支持工单创建、转派、完成、附件上传、内部备注、角色访问控制和审计日志 。可直接对应文档中社区端"工单与协同"模块（4.4.3）的全部需求，也可作为独立服务嵌入系统 。 [github](https://github.com/aottr/paw)

***

## 前端管理看板

**推荐：`Vue Vben Admin`**

基于 Vue3 + Vite + TypeScript + Ant Design Vue 的开源管理后台模板，内置国际化、主题配置，适合快速搭建社区数据看板、辖区老人列表、统计报表等页面 。 [madewithvuejs](https://madewithvuejs.com/vue-vben-admin)

***

## 推荐技术栈总览

| 模块 | 开源项目 | 语言/框架 |
|---|---|---|
| 后端 API + RBAC | `django-vue3-admin` 或 `fastapi-vue-admin` | Python (Django/FastAPI) |
| 前端管理页面 | `Vue Vben Admin` | Vue3 + TypeScript |
| 通话语音识别 | `openai/whisper` | Python |
| 短信 NLP 风险检测 | Hugging Face `transformers` + `bert-base-chinese` | Python |
| 诈骗检测训练数据 | `TeleAntiFraud-28k` | — |
| 实时通知推送 | FastAPI + Redis Streams + WebSocket | Python |
| 工单协同系统 | `paw` | Python (Django) |
| 数据库 | PostgreSQL + Redis | — |

建议以 `django-vue3-admin` 作为基座快速启动 V1.0（账号认证、RBAC、管理后台），再接入 Whisper + Transformers 构建风险识别微服务，工单模块可直接复用 `paw` 的逻辑，降低重复开发成本。
